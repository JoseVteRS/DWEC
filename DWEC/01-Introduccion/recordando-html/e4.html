URL
Un URL (Localizador Uniforme de Recursos, del inglés Uniform Resource Locator) es una cadena de caracteres que sirve para identificar la ubicación de un recurso en Internet y especificar el mecanismo para acceder a él.
HTTP
HTTP (HyperText Transfer Protocol, o Protocolo de Transferencia de Hipertexto) es un protocolo de la capa de aplicación que permite la comunicación entre cliente y servidor en la Web.
TCP/IP
TCP/IP es el conjunto (o “familia”) de protocolos sobre el que se basa Internet, organizado en capas para manejar el envío de datos, direccionamiento, encaminamiento y transporte fiable entre nodos.
Internet
Internet es una red global de redes de computación interconectadas que usan protocolos estándar (principalmente TCP/IP) para comunicarse entre sí. Funciona como infraestructura que permite servicios como la Web, correo electrónico, transferencia de archivos, etc.
W3C
El W3C (World Wide Web Consortium) es una organización internacional que desarrolla estándares para la Web con el objetivo de asegurar su interoperabilidad, accesibilidad y crecimiento coherente.
HTML
HTML (HyperText Markup Language, "Lenguaje de Marcado de Hipertexto") es el lenguaje principal utilizado para estructurar y presentar contenido en páginas web. Define etiquetas (como <p>, <a>, <div>, etc.) que organizan textos, imágenes, enlaces y otros elementos multimedia en la página.
